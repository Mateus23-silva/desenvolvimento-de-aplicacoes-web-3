# Desenvolvimento de Aplicações Web 3

Repositório destinado às atividades desenvolvidas na disciplina **Desenvolvimento de Aplicações Web 3**.

Os projetos foram desenvolvidos com o objetivo de praticar conceitos de **React, manipulação de formulários, componentes e interfaces web**.

---

# Atividades

## 1 - Cadastro de Disciplina

Aplicação desenvolvida em **React** que permite realizar o cadastro de uma disciplina através de um formulário.

### Campos do formulário

* Nome da disciplina
* Descrição da disciplina
* Curso
* Período
* Professor
* Quantidade de alunos

### Funcionamento

Após o preenchimento e envio do formulário, a aplicação exibe um **modal** contendo todas as informações inseridas pelo usuário.

Localização do projeto:

```
/cadastro-disciplina
```

---

## 2 - Sistema de Professores (SPA)

Aplicação **Single Page Application (SPA)** desenvolvida utilizando **React + Vite**.

O sistema permite visualizar professores, suas disciplinas e enviar mensagens através de um formulário de contato.

### Funcionalidades

* Página inicial
* Menu de navegação
* Listagem de disciplinas por professor
* Formulário de contato
* Exibição dos dados enviados em um modal
* Navegação dinâmica sem recarregar a página

Localização do projeto:

```
/sistema-professores
```
