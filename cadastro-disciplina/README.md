# Cadastro de Disciplina

Projeto desenvolvido para a disciplina **Desenvolvimento de Aplicações Web 3**.

A aplicação consiste em um **formulário de cadastro de disciplina** desenvolvido utilizando **React**.

## Objetivo

Praticar a criação de componentes em React e manipulação de formulários, exibindo os dados informados pelo usuário após o envio do formulário.

## Campos do formulário

O formulário possui os seguintes campos obrigatórios:

* Nome da disciplina
* Descrição da disciplina
* Curso
* Período
* Professor
* Quantidade de alunos

## Funcionamento

Após o preenchimento dos campos e submissão do formulário:

* Os dados da disciplina são processados
* Um **modal é exibido na tela**
* O modal apresenta todas as informações inseridas no formulário

## Tecnologias utilizadas

* React
* JavaScript
* HTML
* CSS
