# Sistema de Professores (SPA)

Projeto desenvolvido para a disciplina **Desenvolvimento de Aplicações Web 3**.

O sistema consiste em uma **Single Page Application (SPA)** construída com **React e Vite**, onde é possível visualizar professores, suas disciplinas e enviar mensagens através de um formulário de contato.

---

# Funcionalidades

* Página inicial
* Menu de navegação entre professores
* Listagem de disciplinas por professor
* Formulário de contato
* Exibição dos dados enviados em um modal
* Navegação dinâmica sem recarregar a página (SPA)

---

# Tecnologias utilizadas

* React
* Vite
* JavaScript
* HTML
* CSS
* Node.js

---

# Como executar o projeto

Clone o repositório:

```
git clone https://github.com/Mateus23-silva/desenvolvimento-aplicacoes-web-3.git
```

Entre na pasta do projeto:

```
cd sistema-professores
```

Instale as dependências:

```
npm install
```

Execute o projeto:

```
npm run dev
```

A aplicação estará disponível em:

```
http://localhost:5173
```
